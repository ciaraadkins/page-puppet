How to use:

  1. Click "Start Voice Control" or use keyboard shortcut
  2. Hover over any element on the page
  3. Speak your command naturally
  4. Watch the element change instantly

Example Commands:

  • "Make it red" - Changes color
  • "Make it bigger" - Increases size
  • "Hide it" - Hides the element
  • "Make it transparent" - Changes opacity
  • "Add a shadow" - Adds drop shadow
  • "Rotate it" - Rotates element
